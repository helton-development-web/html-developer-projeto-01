O que é HTML?

Retornar

Na definição do site wikiHow 1 HTML é a abreviatura de Hyper Text Markup Language (Linguagem de Marcação de Hipertexto) e é o código, ou a "linguagem, que é usado para a criação de sites. Pode parecer um pouco assustador se você nunca fez qualquer codificação antes, mas tudo o que você precisa para tentar é de um programa de edição de textos comum e de um navegador de internet. Você pode até reconhecer alguns HTML usados para alterar o texto em fóruns on-line, perfis on-line personalizados ou artigos do wikiHow.

O HTML é uma ferramenta útil para quem usa a internet, e aprender o básico pode levar menos tempo do que se pensava. O blog do site kinsta.com 2 se refere a HTML significa HyperText Markup Language (Linguagem de marcação HyperText).

É a linguagem usada para exibir documentos e várias criações em um navegador padrão. É usado por empresas em todo o mundo para alimentar aplicativos, softwares e sites móveis - quase 92,8% de todos os sites. O aprendizado de HTML abrirá inúmeras oportunidades para você e sua empresa. Você será capaz de codificar sites e ajustar scripts através da compreensão do HTML básico. A princípio, escrever em HTML pode parecer complexo. Dê uma olhada nas linhas de código HTML escritas neste editor de HTML: HTML é uma linguagem de codificação única que requer conhecimento avançado.

Pode não fazer sentido para você agora, mas uma vez que você faça um curso de HTML, você começará a obtê-lo em breve. Conforme você aprende e entende a estrutura necessária do HTML, você será capaz de se lembrar dos atalhos HTML em pouco tempo. No exemplo acima, você pode ver que o código é escrito para uma página HTML, incluindo um cabeçalho, título e corpo. As tags dentro dos parênteses começam a dar a estrutura do documento, enquanto o conteúdo real pode ser adicionado entre eles. Coletivamente, as tags e o conteúdo são considerados um elemento. O primeiro passo para entender HTML é aprender todas as tags mais comumente usadas e o que elas fazem.

Por exemplo, a tag p denota um parágrafo. A ordem e a forma como você coloca as tags são como a estrutura e o estilo da sua página web ou ativo ganharão vida. Algumas das tags mais populares incluem:
  • H1 - primeiro tamanho do texto do cabeçalho
